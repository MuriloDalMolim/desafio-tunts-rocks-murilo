## Requirements

- node >= 20

- Google Cloud Platform Service Account

- Git

## Instructions

- Create a .env file in the project root directory containing these variables:
- Both must be filled in with the keys obtained with the creation of the Google Cloud Platform Service Account
```bash
GOOGLE_SERVICE_ACCOUNT_EMAIL=
GOOGLE_PRIVATE_KEY=
```


### Commands

Dependencies installation
```bash
npm install
```

Application execution
```bash
npm start
```
